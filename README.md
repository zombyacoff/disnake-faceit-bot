# FACEIT Discord Bot

The official showcase example for the [faceit-python](https://github.com/zombyacoff/faceit-python/) library, built using [disnake](https://github.com/DisnakeDev/disnake).

## Features

- **Lifetime Stats**: Get a comprehensive overview of a player's performance.
- **Recent Performance**: Analyze average statistics over a specific number of recent matches.

## Commands

- `/stats <nickname>` — Retrieves and displays lifetime statistics for the given FACEIT player.
- `/recent <nickname> [matches]` — Calculates and displays average statistics for a specified number of recent matches (supports a range from 1 to 100).

## Requirements

- Python 3.10+
- A FACEIT API Key
- A Discord Bot Token
